Browser Support

WebRx is tested on the following browsers:

  • Mozilla Firefox: 5 - current
  • Google Chrome: 5 - current
  • Microsoft Internet Explorer: 9 - current
  • Apple Safari for Mac OS: 5 - current
  • Apple Safari for iOS: 6 - current
  • Opera: current
